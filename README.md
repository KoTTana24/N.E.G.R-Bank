# N.E.G.R-Bank

## Reliable Single Global Reserve
## Надёжный Единый Глобальный Резерв

## How to install N.E.G.R Bank
### Linux & MacOS
Install python3

Install PySide6 with pip

```sh
python -m venv env
source env/bin/activate
pip install pyside6
```

Install PySide6 with pacman

```sh
sudo pacman -S pyside6
```

Install N.E.G.R Bank
```sh
git clone https://github.com/KoTTana24/N.E.G.R-Bank
cd N.E.G.R-Bank
python3 main.py
```

### Windows
Install python3

Install PySide6 with pip
```sh
python -m venv venv
venv\Scripts\activate
pip install PySide6

```

Clone repo and 
```sh
python3 main.py
```
